Cara Membuat API Key Google Cloud Console

Langkah 1: Create Project

Buka Google Cloud Console dan login menggunakan akun Google Anda. Pada halaman Dashboard, klik “Select a project” di bagian atas, kemudian pilih “New Project”. Berikan nama untuk proyek Anda dan klik “Create”.

Langkah 2: Activate Library

Kunjungi halaman Generative Language API dan pilih proyek yang baru Anda buat. Klik “Enable” untuk mengaktifkan library tersebut pada proyek Anda.

Langkah 3: Buka Credential, lalu Create Apikey

Kunjungi halaman Credentials dan klik “Create Credentials” di bagian atas. Pilih “API key” dari dropdown menu. Setelah API key dibuat, Anda bisa melihat dan menyalin API key tersebut untuk digunakan pada aplikasi Anda.
