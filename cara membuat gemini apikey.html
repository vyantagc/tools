Panduan Membuat Google Gemini API Key

Langkah 1: Buat Project Google Cloud

  1. Buka Google Cloud Console.
  2. Login menggunakan akun Gmail atau G Suite Anda.
  3. Klik pemilih proyek di bagian atas navigasi dan pilih Project Baru.
  4. Isi nama project, organisasi, dan lokasi sesuai kebutuhan, lalu klik Buat.

Langkah 2: Aktifkan Library Gemini AI dan YouTube API

Aktifkan API yang diperlukan untuk project:

  1. Buka Library Gemini AI API dan klik Enable.
  2. Buka Library YouTube Data API dan klik Enable.

Langkah 3: Buat API Key

  1. Di halaman Kredensial, klik Buat Kredensial dan pilih API Key.
  2. Salin API key yang dihasilkan dan simpan dengan aman.
  3. Opsional, batasi API key ke IP tertentu, referer, atau API untuk keamanan tambahan:
    • Klik Edit API Key.
    • Atur pembatasan penggunaan di bawah Application Restrictions dan API Restrictions.
    • Klik Simpan.

Catatan Penting

  • Gunakan satu akun Gmail atau G Suite untuk membuat beberapa project cloud.
  • Pastikan setiap project cloud memiliki API key unik untuk menghindari konflik kuota.
